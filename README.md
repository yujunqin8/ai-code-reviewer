## ✨ 特性

- 🤖 **多模型支持** - 支持 OpenAI GPT-4、Claude、本地 LLM
- 📁 **多格式支持** - Python、JavaScript、TypeScript、Go、Rust 等
- 🎯 **智能分析** - 代码质量、安全漏洞、性能优化建议
- 🔄 **CI/CD 集成** - GitHub Actions、GitLab CI、Jenkins 支持
- 📊 **详细报告** - 美观的 Markdown/HTML 报告输出
- ⚡ **极速响应** - 增量审查，只分析变更代码

## 🚀 快速开始

### 安装

```bash
pip install ai-code-reviewer
```

### 使用

```bash
# 审查单个文件
ai-review review src/main.py

# 审查 Git 变更
ai-review diff

# 审查 PR
ai-review pr --repo owner/repo --number 123
```
